# Frontend Cine (React + Vite)

Proyecto frontend listo para conectarse al backend:
`https://taller-patrones-h3l7.vercel.app`

## Pasos para ejecutar localmente
1. Descomprime el zip.
2. En la carpeta del proyecto ejecuta:
   ```bash
   npm install
   npm run dev
   ```
3. Abre http://localhost:5173
